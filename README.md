Debug .NET Core Docker Application

Requirements
* Visual Studio Code
* C# Extension
* Docker Tools for Mac / Windows
* dotnet CLI RC3 

1. clone repo
2. `rm -rf .git`
3. `dotnet restore -r debian.8-x64`
4. open vs code - `code .`
5. set break point in code.
6. open debugger and click the "play" button
